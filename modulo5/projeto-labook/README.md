## Labook

Projeto Labook foi feito com o intuito de simular uma API de rede social.

### Funcionalidades

- Cadastrar usuário
- Login
- Feed do usuário
- Fazer amizade
- Desfazer amizade

- Criar post
- Criar comentário
- Curtir post
- Descurtir post
- Pegar todos os posts pelo tipo
- Pegar post pelo id

### Tecnologias utilizadas

- Typescript
- Node.js
- Knex
- MySQL
- Express
- Dotenv
- Cors
- UUID
- Bcrypt
- JWT
